## 📌 Objetivo
[cite_start]O objetivo principal desta atividade é praticar a estruturação de códigos utilizando **Funções** em Python para resolver problemas de lógica, cálculos e manipulação de listas[cite: 8].

## 🚀 Desafios Inclusos

### 1. Calculadora com Funções
[cite_start]Criação de um sistema de calculadora onde cada operação matemática (soma, subtração, multiplicação e divisão) é tratada por uma função específica[cite: 9].

### 2. Analisador de Números (Pares e Ímpares)
O programa recebe uma quantidade de números definida pelo usuário e os armazena em uma lista. [cite_start]Uma função é responsável por identificar e exibir a quantidade e os valores de números pares e ímpares presentes na lista[cite: 10, 11, 12, 13].

### 3. Monitoramento de Temperatura (INMET)
Simulação de análise de dados meteorológicos baseada em 8 capturas diárias de temperatura (das 06h às 00h). O código calcula a média e classifica a situação térmica em:
* [cite_start]**Baixa:** < 20°C [cite: 19]
* [cite_start]**Ideal:** Entre 20°C e 35°C [cite: 20]
* [cite_start]**Em Alerta:** Entre 36°C e 50°C [cite: 21]
* [cite_start]**Risco:** > 50°C [cite: 21, 22]

### 4. Análise de pH Laboratorial
Algoritmo para auxiliar um químico na classificação de 12 produtos. A função recebe uma lista de valores de pH e os organiza em três categorias distintas:
* [cite_start]**Ácido:** pH < 7 [cite: 27]
* [cite_start]**Neutro:** pH = 7 [cite: 28]
* [cite_start]**Básico:** pH entre 7 e 14 [cite: 29]
